# BEAR UI

**基于Vue3+TypeScript的一个可维护性较高的组件库**

## 安装

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 bearUI, 然后您就可以使用打包工具，例如 Vite 和 webpack

### 选择一个你喜欢的包管理器

#### NPM

```bash
$ npm install bear-ui --save
```

#### Yarn

```bash
$ yarn add bear-ui
```

#### Pnpm

```bash
$ pnpm install bear-ui
```
