# 安装

## 环境支持

由于 Vue 3 不再支持 IE11，BearUI 也不再支持 IE11 浏览器。

## 版本

BearUI 目前还在开发迭代中

## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 bearUI, 然后您就可以使用打包工具，例如 Vite 和 webpack

### 选择一个你喜欢的包管理器

#### NPM

```bash
$ npm install bear-ui --save
```

#### Yarn

```bash
$ yarn add bear-ui
```

#### Pnpm

```bash
$ pnpm install bear-ui
```

## 浏览器直接引入
